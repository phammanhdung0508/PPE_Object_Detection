## 2025-05-15 - High-Contrast Focus Indicators in Dark Dashboards
**Learning:** In dark-themed, dense dashboards, standard focus outlines often lack sufficient contrast. A multi-layered focus ring using both `outline` and `box-shadow` (e.g., a primary brand color ring followed by a secondary light ring) ensures visibility across different button variants and prevents the indicator from blending into the background. Additionally, in grid-based layouts, focus indicators must have a higher `z-index` to avoid being clipped or obscured by adjacent tiles or absolute-positioned headers.
**Action:** Use a CSS pattern like `:focus-visible { outline: 2px solid var(--text-primary); outline-offset: 2px; box-shadow: 0 0 0 2px var(--bg-base), 0 0 0 4px var(--color-brand); z-index: 20; }` for all interactive dashboard elements.
